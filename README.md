This project features an interactive Power BI dashboard designed to track patient visits, wait times, and department referrals using real hospital-like datasets. It provides clear and actionable insights through dynamic visuals and filters.

Tools Used
Python
DataBase
Power BI
Power Query
DAX

Features
Patient visit tracking
Wait time analysis
Department-wise referrals
Filters by age, gender, department, and rating
KPIs generated using DAX

Files Included
CareTrackDashboard.pbix
dataset.xlsx (sample dataset)
/screenshots (dashboard images)

How to Run
Download the .pbix file
Open in Power BI Desktop
Load the dataset
Refresh to view KPIs
